# Version 0.3.0 — Design Baseline

## Overview

This version establishes the visual baseline for Parking Yetu.
All future visual changes must be compared against this version.

## Status

- ✅ Visual identity restored
- ✅ Moving transport background
- ✅ Glass effect on auth box
- ✅ Dark/light theme toggle
- ✅ Responsive scaling
- ✅ Accessibility support

## Frozen Elements

- Background opacity: 0.03 (desktop), 0.025 (tablet), 0.02 (mobile)
- Font stack: Orbitron (headings), Inter (body)
- Colors: Purple (#6C3CE1), Cyan (#00D4FF), Green (#00FF88), Amber (#FF6B35), Red (#FF2D55)
- Card radius: 32px
- Button radius: 16px
- Spacing: 4, 8, 12, 16, 24, 32, 48, 64
