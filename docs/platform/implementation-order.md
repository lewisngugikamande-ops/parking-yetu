# Implementation Order

## Phase 0: Foundation (Week 1)
- Authentication (JWT)
- Multi-tenant middleware
- Organization context
- Audit logging

## Phase 1: Commercial Layer (Week 1-2)
- Lead management
- Application form
- Application review workflow
- Customer management
- Subscription plans
- Partner model

## Phase 2: Access Engine (Week 2-3)
- Credential model and QR generation
- Identity resolution
- Access requests
- Policy evaluation
- Session management

## MVP Priority (2-3 Weeks)
1. Authentication
2. Multi-tenant
3. Organization management
4. User management
5. QR credentials
6. Access requests
7. Simple policies
8. Basic admin UI
