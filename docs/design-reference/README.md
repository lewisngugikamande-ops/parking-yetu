# Parking Yetu — Visual References

## Purpose

These images serve as the visual baseline for Parking Yetu.
Every UI change should be compared against these references.

## Files

- `login-desktop.png` — Login screen on desktop
- `login-tablet.png` — Login screen on tablet
- `login-mobile.png` — Login screen on mobile
- `dark-theme.png` — Dark theme
- `light-theme.png` — Light theme

## How to Use

1. Before making UI changes, review these references
2. After making changes, compare the result
3. If they differ significantly, revisit the change

## Update Process

When the design intentionally evolves:
1. Update these screenshots
2. Document the change in `releases/version-x.md`
3. Update the design system
