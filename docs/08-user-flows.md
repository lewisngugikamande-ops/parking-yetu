# Parking Yetu - User Flows

## Vehicle Entry (Self Check-in)
1. User navigates to Entry page
2. User enters vehicle details
3. User clicks Check In
4. System validates and creates ParkingSession
5. QR code is generated and displayed

## Vehicle Exit (Security)
1. Security logs in
2. Security finds vehicle (QR or manual)
3. System displays vehicle details
4. Security reviews and selects action
5. System processes exit and creates audit log

## User Creation (Admin)
1. Admin navigates to Admin page
2. Admin opens User Management
3. Admin enters user details
4. Admin clicks Create User
5. System creates account and Firestore document
